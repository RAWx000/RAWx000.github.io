ЭКСКЛЮЗИВНЫЕ РАЗРАБОТКИ
Для заказчика разработаны утолщенные вязаные
пледы из шерсти, выполнены по технологии
зеркальной жаккардовой вязки, экстра-большой
размер, пледы дополнил полноцветный вышитый
шеврон с логотипом созданный по
без-соединительной технологии вышивки.
Изюминку к заказу добавили с помощью
дизайнерских портупей из натуральной кожи,
выполненных вручную.
ИНДИВИДУАЛЬНОСТЬ ВО ВСЁМ
Для клиента связаны фактурные пледы из шерсти,
добавлено брендирование шевроном из
натуральной толстой кожи, проведена кропотливая
работа по подбору эксклюзивного корпоративного
цвета заказчика, в наборе разработаны идентичные
по стилистике фактурные шапка и шарф. В роли
упаковки выступила атласная лента с нанесением
логотипа методом фольгирования.
НЕПОВТОРИМЫЕ ТЕХНОЛОГИИ
Эксклюзивные решения по брендированию
пледов. Логотип на гигантском пледе исполнен
в редчайшей и ультра-сложной технике вязания
обратными петлями, переходящими в фактурный
узор косы.
VIP ПОДАРКИ СОТРУДНИКАМ
Вязаные пледы из натуральной шерсти,
индивидуальный жаккардовый узор в котором
отражены ценности предприятия, а также
приуроченный к памятной дате, утолщенный
формат изделия категории люкс. Объем 7000
штук произведен в кратчайшие сроки
и доставлен на завод к закачику.
ПОДАРКИ ДЛЯ ПАРТНЕРОВ
Вязаные пледы из натуральной шерсти мериноса
размера «стандарт», реализована идея и дизайн
заказчика, сам плед имеет структурную вязку,
а по краям гладкий и цветной узор с слоганом
компании, выполнен подбор индивидуальных
цветов.
ПОДАРКИ НА ПРАЗДНИК
Совместно с заказчиком разработали концепт
корпоративных подарков к Новому году, выполнен
подбор цвета по системе Pantone, связаны
фактурные пледы с логотипом, исполненном
в структуре самого пледа. Под пледы вырублена
крафтовая коробка специального размера с окном
и картонной обечайкой с логотипом, а также
поздравлением от компании.
